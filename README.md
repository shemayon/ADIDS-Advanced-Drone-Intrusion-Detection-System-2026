# A-DIDS: Advanced Drone Intrusion Detection System (2025-2026)

This directory contains the upgraded, modularized version of the TSLT-Net project, featuring Advanced AI/ML capabilities for the 2026 UAE Defence showcase.

## Directory Structure

- `data/`: ISOT Drone Anomaly Detection Dataset (CSV format).
- `models/`: Saved model weights (.pth, .h5) and ONNX/TensorRT exports.
- `modules/`: Modularized Python components for the A-DIDS engine.
- `notebooks/`: Showcase and experimental Jupyter notebooks.
- `scripts/`: Data preprocessing, model conversion, and deployment utilities.
- `tests/`: Project-wide verification tests (pytest).
- `config/`: Configuration files and tactical edge constants.
